# BlackRice Tarot - iOS 移动端 UI/UX 重新设计计划

> 版本：2.0  
> 创建日期：2026-03-11  
> 设计风格：Premium Dark + Holographic Luxury  
> 优先级：iOS 移动端优先  
> 核心特性：Tilt + Holo 卡牌效果 | Daily Fortune | 丝滑动效

---

## 一、设计愿景

### 1.1 核心理念

**"奢华神秘，触手可及"** — 借鉴 Gold Tarot、Tarot Grand Luxe 等顶级塔罗应用，打造沉浸式高级体验。

**三大设计支柱：**

| 支柱 | 描述 | 体现 |
|------|------|------|
| **卡牌至上** | 卡牌是绝对主角 | 全屏展示、Tilt + Holo 效果、环境光反射 |
| **丝滑动效** | 每个交互都有反馈 | Spring 物理动画、手势跟随、60fps |
| **每日仪式** | Daily Fortune 核心入口 | 首屏突出、收集激励、通知推送 |

### 1.2 设计灵感

| 参考应用 | 学习要点 |
|----------|----------|
| **Gold Tarot** | 金色反光卡牌效果、环境光映射、丝滑翻牌 |
| **Tarot Grand Luxe** | Ciro Marchetti 高清卡面、洗牌切牌动画 |
| **Co-Star** | 极简排版、沉浸式体验、每日通知 |
| **The Pattern** | 个性化内容、优雅渐变、社交分享 |

### 1.3 设计原则

| 原则 | 描述 | 实现方式 |
|------|------|----------|
| **卡牌为王** | 卡牌是视觉焦点 | 全屏展示、最小 UI 干扰 |
| **光影流动** | Holo 效果随手势变化 | hover-tilt + 光线追踪感 |
| **物理反馈** | 动效符合物理直觉 | Spring 动画、惯性滑动 |
| **仪式氛围** | 每个操作都有仪式感 | 进场动画、声音、震动 |

```

         ├── 高清牌面图像
         └── 文字信息

         ├── 金色边缘光晕

         ├── 柔和阴影
         └── 跟随倾斜方向
```

#### 卡牌状态机

```typescript
type CardState = 
  | 'idle'      // 静止 - 微弱呼吸光效
  | 'hover'     // 悬浮 - 上浮 + 阴影加深
  | 'tilting'   // 倾斜 - Tilt 跟随手势
  | 'flipping'  // 翻转 - 3D 旋转动画
  | 'revealed'  // 揭示 - Holo 全息激活
  | 'zooming'   // 放大 - 居中 + 缩放
  | 'zoomed'    // 已放大 - 继续 Tilt
```

## 三、动效系统 — 丝滑体验

### 3.1 动效原则

| 原则 | 描述 | 实现 |
|------|------|------|
| **物理真实** | 动效符合物理直觉 | Spring 弹簧动画 |
| **60fps** | 保持流畅无卡顿 | GPU 加速、will-change |
| **有反馈** | 每个操作都有响应 | 触觉 + 视觉 + 声音 |
| **可中断** | 动画可被新交互打断 | 动画状态机 |

### 3.2 Spring 预设

```typescript
const springPresets = {
  // 快速响应 - 按钮、微交互
  snappy: { 
    stiffness: 400, 
    damping: 30,
    mass: 1 
  },
  
  // 平滑过渡 - 页面、卡片
  smooth: { 
    stiffness: 200, 
    damping: 25,
    mass: 1 
  },
  
  // 弹性反馈 - 翻牌、放大
  bouncy: { 
    stiffness: 300, 
    damping: 20,
    mass: 1 
  },
  
  // 慢速优雅 - 进场、仪式
  gentle: { 
    stiffness: 100, 
    damping: 20,
    mass: 1 
  }
}
```

### 3.3 动效时序表

| 动作 | 时长 | 曲线 | 备注 |
|------|------|------|------|
| 按钮按下 | 100ms | ease-out | 缩放 0.97 |
| Tab 切换 | 250ms | spring(smooth) | 图标缩放 |
| 页面转场 | 350ms | ease-in-out | iOS 风格 |
| 卡牌入场 | 500ms | spring(bouncy) | 每张间隔 150ms |
| 翻牌 | 700ms | cubic-bezier | 3D 旋转 |
| 放大居中 | 800ms | spring(smooth) | translate + scale |
| Holo 激活 | 500ms | ease-out | glare 淡入 |

### 3.4 触觉反馈 (Haptic)

| 场景 | 类型 | iOS API |
|------|------|---------|
| 按钮点击 | Light | UIImpactFeedbackGenerator(.light) |
| 抽牌 | Medium | UIImpactFeedbackGenerator(.medium) |
| 翻牌揭示 | Heavy | UIImpactFeedbackGenerator(.heavy) |
| Tab 切换 | Selection | UISelectionFeedbackGenerator |
