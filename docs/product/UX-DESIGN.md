## 一、设计原则

### 1.1 核心设计理念

#### 🌙 神秘沉浸 (Mystical Immersion)

创造一个让用户脱离日常、进入神秘仪式空间的体验。通过深色背景、金色点缀、星空效果营造占卜的庄重感。

#### ✨ 简约优雅 (Elegant Simplicity)

减少视觉噪音，让用户专注于占卜本身。每个界面只保留必要元素，通过留白和层次创造呼吸感。

#### 🎭 仪式感体验 (Ritualistic Experience)

将占卜过程设计成一场小型仪式：选择、洗牌、抽取、揭示、解读，每个步骤都有明确的节奏和反馈。

#### 💫 情感化设计 (Emotional Design)

通过动画、文案、视觉细节传递温暖和关怀，让用户感受到被理解和被引导。

### 1.2 设计原则矩阵

| 原则 | 表现形式 | 避免做法 |
|------|----------|----------|
| 神秘感 | 深色、金色、渐变、光晕 | 明亮、扁平、普通配色 |
| 简约 | 单一焦点、充足留白 | 信息过载、密集布局 |
| 仪式感 | 分步骤、有节奏的动画 | 一步到位、跳过过程 |
| 情感化 | 温暖文案、柔和动效 | 冷冰冰的系统提示 |

## 四、动效设计规范

### 4.1 动效原则

| 原则 | 说明 | 参数 |
|------|------|------|
| 自然流畅 | 使用缓动函数 | ease-out, spring |
| 有意义 | 动效服务于交互 | 不纯粹装饰 |
| 适度 | 不过度打断用户 | 300-500ms |
| 可控 | 尊重减少动效偏好 | prefers-reduced-motion |

### 4.2 核心动效清单

#### 页面转场

```typescript
// 页面淡入淡出
{
  enter: { opacity: [0, 1], duration: 250 },
  leave: { opacity: [1, 0], duration: 250 }
}
```

#### 卡牌动效

| 动效 | 触发 | 参数 |
|------|------|------|
| 入场 | 抽牌后 | scale 0.8→1, y 30→0, 400ms |
| 翻牌 | 点击卡牌 | rotateY 0→180°, 600ms |
| 悬停 | 鼠标悬停 | translateY -8px, 200ms |
| 退场 | 重新抽牌 | scale 1→0.8, opacity 1→0, 300ms |

#### 翻牌详细时序

```
0ms     - 开始翻转
0-300ms - 牌背旋转 0° → 90°
300ms   - 切换显示牌面
300-600ms - 牌面旋转 90° → 180°
600ms   - 完成翻转
```

### 4.3 微交互

| 元素 | 交互 | 反馈 |
|------|------|------|
| 按钮 | Hover | 亮度提升 + 光晕 |
| 按钮 | Click | 轻微缩放 (0.95) |
| 卡牌 | Hover | 上浮 8px |
| 导航项 | Active | 背景色变化 |
| 星星 | 持续 | 闪烁动画 3s循环 |

### 4.4 动效时长规范

| 类型 | 时长 | 场景 |
|------|------|------|
| 快速 | 100-200ms | 按钮状态、悬停 |
| 标准 | 200-400ms | 元素入场、转场 |
| 慢速 | 400-600ms | 翻牌、展开 |
| 持续 | 2-5s | 背景动画、循环 |

## 六、交互设计详解

### 6.1 占卜流程交互

#### 步骤一：牌阵选择

```
状态：未选择 → 选中
视觉：描边 → 填充 + 光晕
反馈：选中动画 (scale 1.05 → 1)
限制：抽牌后禁用切换
```

#### 步骤二：抽牌

```
点击 "开始抽牌" 按钮
↓
按钮消失（淡出）
↓
卡牌依次入场（间隔100ms）
↓
显示提示 "点击卡牌翻开查看"
```

#### 步骤三：翻牌

```
点击未翻开的卡牌
↓
3D翻转动画（600ms）
↓
显示牌面（符号 + 名称 + 位置标签）
↓
检查是否全部翻开
↓
全部翻开时显示 "查看解读" 按钮
```

#### 步骤四：查看解读

```
点击 "查看解读 →"
↓
页面转场（淡出淡入）
↓
解读内容依次入场（间隔150ms）
↓
滚动浏览各牌解读
↓
最后显示综合指引
```

### 6.2 手势与触控

| 手势 | 区域 | 响应 |
|------|------|------|
| 点击 | 卡牌 | 翻牌 |
| 点击 | 按钮 | 执行操作 |
| 滑动 | 解读页 | 滚动内容 |
| 长按 | 牌库卡牌 | 预览详情 (V2) |

### 6.3 键盘导航

| 按键 | 功能 |
|------|------|
| Tab | 焦点切换 |
| Enter | 激活当前焦点元素 |
| Space | 翻牌 / 按钮点击 |
| Esc | 关闭弹窗 / 返回 |
| ← → | 牌阵选择切换 |

## 八、特殊状态设计

### 8.1 加载状态

```
视觉：金色脉动环 + "宇宙正在为你准备答案..."
持续：最少800ms（避免闪烁）
位置：屏幕中央
```

### 8.2 空状态

```
视觉：淡化的占位图标 + 引导文案
动效：轻微上下浮动
交互：包含行动按钮
```

### 8.3 错误状态

```
视觉：温和的提示（非红色警告）
文案：友好、可操作
交互：提供重试选项
```
