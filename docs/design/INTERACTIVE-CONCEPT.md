## 二、核心页面设计

### 2.1 Daily Fortune 页（沉浸式单卡）

**设计理念**：神秘、仪式感、第一印象

```
┌──────────────────────────────────────────┐
│                                          │
│           ✦ 粒子星空背景 ✦              │
│                                          │
│                                          │
│         ╔═════════════════════╗         │
│         ║                     ║         │
│         ║                     ║         │
│         ║    🔮 牌背面       ║  ← 大尺寸卡牌
│         ║   (悬浮 + 呼吸光效) ║     居中展示
│         ║                     ║
│         ║                     ║         │
│         ╚═════════════════════╝         │
│                                          │
│                                          │
│        "轻触，开启今日指引"             │  ← 引导文案
│                                          │
│              (微弱脉冲)                  │
│                                          │
└──────────────────────────────────────────┘
无导航栏，全沉浸式
```

**交互流程**：
1. 进入页面 → 卡牌从下方飘入
2. 卡牌悬浮 + 微弱呼吸光效
3. 点击卡牌 → 华丽翻牌动画 (3D 旋转 + 金色边缘光)
4. 翻牌后 → 卡牌上方展示牌名和简短含义
5. 2秒后 → 黑米猫咪从下方滑入

## 三、收集系统设计

### 3.1 牌库收集

**解锁机制**：
- 每次抽到新牌 → 解锁该牌在牌库中的展示
- 初始状态：所有牌面模糊 + 锁定
- 解锁后：可查看高清牌面 + 详细牌义

```
┌──────────────────────────────────────────┐
│  ✦ 牌库 ✦                    22/78     │
│                                          │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐           │
│  │ 🔓 │ │ 🔓 │ │ 🔒 │ │ 🔒 │           │
│  │愚者│ │魔术│ │ ?? │ │ ?? │           │
│  └────┘ └────┘ └────┘ └────┘           │
│                                          │
│  🔓 已解锁: 22张                        │
│  🔒 待解锁: 56张                        │
│                                          │
└──────────────────────────────────────────┘
```

### 3.2 皮肤系统

**皮肤类型**：
- 牌背皮肤 (神秘紫、森林绿、樱花粉...)
- 全息效果预设 (标准、彩虹、星空...)
- 背景主题 (星空、深海、森林...)

**解锁方式**：
- 累计占卜次数
- 连续登录天数
- 特定成就

## 四、技术实现要点

### 4.1 黑米猫咪实现方案

**方案 A: CSS 3D + 动画 (轻量)**
```css
/* 伪3D 猫咪 - CSS 实现 */
.cat-character {
  transform: perspective(1000px) rotateY(var(--cat-rotation));
  transition: transform 0.3s ease;
}

.cat-character:hover {
  --cat-rotation: 5deg;
}

/* 眨眼动画 */
@keyframes blink {
  0%, 100% { --eye-scale: 1; }
  50% { --eye-scale: 0.1; }
}
```

**方案 B: Live2D / Spine (高级)**
- 完整的 2D 骨骼动画
- 更丰富的表情和动作
- 需要额外资源制作

**方案 C: Lottie 动画 (中等)**
- After Effects 导出
- 较丰富的动画
- 文件较小

### 4.2 状态管理

```typescript
// 用户每日状态
interface DailyState {
  hasDrawnDaily: boolean      // 今日是否已抽取 daily fortune
  dailyCard: DrawnCard | null // 今日 daily 卡牌
  totalReadings: number       // 累计占卜次数
  unlockedCards: string[]     // 已解锁牌面 ID
  unlockedSkins: string[]     // 已解锁皮肤 ID
}

// 每日重置逻辑
function checkDailyReset() {
  const today = new Date().toDateString()
  if (lastVisitDate !== today) {
    dailyState.hasDrawnDaily = false
    dailyState.dailyCard = null
  }
}
```

## 五、MVP 优先级
