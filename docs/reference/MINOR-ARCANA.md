# 小阿尔卡纳 (Minor Arcana)

> 共56张牌，分为4个花色，每个花色14张（1-10 + 4张宫廷牌）  
> 基于韦特塔罗（Rider-Waite Tarot）标准  
> 更新日期：2026-03-11

---

## 概述

小阿尔卡纳（Minor Arcana）反映日常生活的各个层面，与大阿尔卡纳的"重大主题"相比，更关注具体事件和日常经历。
