# 大阿尔卡纳 (Major Arcana)

> 共22张牌，代表人生的重大主题和精神旅程  
> 基于韦特塔罗（Rider-Waite Tarot）标准  
> 更新日期：2026-03-11

---

## 概述

大阿尔卡纳（Major Arcana）是塔罗牌中的 22 张牌，代表人生中的重大主题、转折点和灵性课题。「Arcana」源自拉丁语，意为「秘密」或「神秘」。

### 愚者之旅 (The Fool's Journey)

大阿尔卡纳讲述了「愚者」从天真无知到智慧圆满的灵性成长之旅：

```
愚者(0) → 魔术师(1) → ... → 世界(21)
  ↑                              ↓
  └──────── 新的循环 ←───────────┘
```

### 按旅程阶段

| 阶段 | 牌 (编号) | 主题 |
|:---:|:---|:---|
| **物质世界** | 愚者(0) → 战车(VII) | 外在世界的探索、社会身份建立 |

| **精神世界** | 恶魔(XV) → 世界(XXI) | 宇宙力量与命运、自我超越 |
