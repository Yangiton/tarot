# 塔罗牌阵 (Spreads) 完整参考

> 更新日期：2026-03-11

---

## 概述

牌阵（Spread）是塔罗牌解读的框架，每个位置代表不同的含义。选择合适的牌阵能帮助更精准地回答问题。

---

## 基础牌阵

### 单牌占卜 (One Card)

最简单的牌阵，适合快速占卜或每日指引。

```
    ┌─────┐
    │     │
    │  1  │
    │     │
    └─────┘
    今日指引
```

| 位置 | 含义 |
|------|------|
| 1 | 今日/当下的核心信息 |

**适用场景**
- 每日一卡
- 快速回答是/否问题
- 当下最需要的建议
- 冥想焦点

---

### 三牌阵 (Three Card Spread)

最常用的基础牌阵，变体丰富。

```
┌─────┐   ┌─────┐   ┌─────┐
│     │   │     │   │     │
│  1  │   │  2  │   │  3  │
│     │   │     │   │     │
└─────┘   └─────┘   └─────┘
  过去       现在       未来
```

#### 常见变体

| 变体 | 位置 1 | 位置 2 | 位置 3 |
|------|--------|--------|--------|
| 时间线 | 过去 | 现在 | 未来 |
| 情况分析 | 情况 | 行动 | 结果 |
| 自我探索 | 思想 | 感受 | 行为 |
| 问题解决 | 问题 | 原因 | 建议 |
| 选择 | 选项A | 不变 | 选项B |
| 机会 | 机会 | 挑战 | 结果 |

**适用场景**
- 一般问题咨询
- 事件发展趋势
- 简单的选择题
- 快速深入分析

---

### 五牌阵 (Five Card Spread)

比三牌阵更详细的分析。

```
          ┌─────┐
          │     │
          │  3  │ 潜意识/过去影响
          │     │
          └─────┘
              
┌─────┐   ┌─────┐   ┌─────┐
│     │   │     │   │     │
│  5  │   │  1  │   │  4  │
│     │   │     │   │     │
└─────┘   └─────┘   └─────┘
 建议       现状       未来
              
          ┌─────┐
          │     │
          │  2  │ 挑战/需要注意
          │     │
          └─────┘
```

| 位置 | 含义 |
|------|------|
| 1 | 现状 - 当前情况的核心 |
| 2 | 挑战 - 需要面对的问题 |
| 3 | 过去 - 影响现状的过去因素 |
| 4 | 未来 - 可能的发展方向 |
| 5 | 建议 - 应该采取的行动 |

**适用场景**
- 中等复杂度问题
- 需要行动建议时
- 理解因果关系

## 进阶牌阵

### 凯尔特十字 (Celtic Cross)

最经典的全面分析牌阵。

```
                    ┌─────┐
                    │     │
                    │ 10  │  最终结果
                    │     │
                    └─────┘
                    
                    ┌─────┐
                    │     │
                    │  9  │  希望与恐惧
                    │     │
                    └─────┘
                    
        ┌─────┐     ┌─────┐
        │     │     │     │
        │  3  │     │  8  │  外部环境
        │     │     │     │
        └─────┘     └─────┘
            
    ┌─────┐ ┌───┐   ┌─────┐
    │     │ │   │   │     │
    │  5  │ │1/2│   │  6  │  近未来
    │     │ │   │   │     │
    └─────┘ └───┘   └─────┘
            
        ┌─────┐     ┌─────┐
        │     │     │     │
        │  4  │     │  7  │  自我态度
        │     │     │     │
        └─────┘     └─────┘
```

| 位置 | 名称 | 含义 |
|------|------|------|
| 1 | 现状 | 当前情况的核心 |
| 2 | 障碍 | 横跨在1上，主要挑战或辅助力量 |
| 3 | 潜意识 | 潜在的影响、不自知的因素 |
| 4 | 过去 | 近期过去的影响 |
| 5 | 意识目标 | 当前的想法、目标、期望 |
| 6 | 近未来 | 即将发生的事 |
| 7 | 自我态度 | 问卜者对情况的态度 |
| 8 | 外部环境 | 他人或环境的影响 |
| 9 | 希望/恐惧 | 内心的期望或担忧 |
| 10 | 最终结果 | 如果继续当前路径的结果 |

**解读顺序**
1. 先看 1+2（核心情况）
2. 看 3-4-5-6（过去-意识-未来）
3. 看 7-8-9（内外因素）
4. 最后综合看 10

**适用场景**
- 重大人生问题
- 需要全面分析
- 复杂的决策
- 深度自我探索

### 关系牌阵 (Relationship Spread)

专门用于分析两人关系。

```
    ┌─────┐           ┌─────┐
    │     │           │     │
    │  1  │           │  2  │
    │     │           │     │
    └─────┘           └─────┘
      你               对方
    
    ┌─────┐           ┌─────┐
    │     │           │     │
    │  3  │           │  4  │
    │     │           │     │
    └─────┘           └─────┘
   你如何看           对方如何看
   这段关系           这段关系
    
          ┌─────┐
          │     │
          │  5  │
          │     │
          └─────┘
         关系核心
          
          ┌─────┐
          │     │
          │  6  │
          │     │
          └─────┘
         关系走向
```

| 位置 | 含义 |
|------|------|
| 1 | 你在关系中的状态 |
| 2 | 对方在关系中的状态 |
| 3 | 你如何看待这段关系 |
| 4 | 对方如何看待这段关系 |
| 5 | 关系的核心/基础 |
| 6 | 关系的未来发展 |

**变体**：可添加第 7 张牌作为建议

### 决策牌阵 (Decision Spread)

帮助在多个选项间做决定。

```
          ┌─────┐
          │     │
          │  1  │  当前情况
          │     │
          └─────┘
              │
    ┌─────────┼─────────┐
    ▼         ▼         ▼
┌─────┐   ┌─────┐   ┌─────┐
│     │   │     │   │     │
│  2  │   │  3  │   │  4  │
│     │   │     │   │     │
└─────┘   └─────┘   └─────┘
选项 A    不行动    选项 B

┌─────┐   ┌─────┐   ┌─────┐
│     │   │     │   │     │
│  5  │   │  6  │   │  7  │
│     │   │     │   │     │
└─────┘   └─────┘   └─────┘
结果 A   不行动结果   结果 B
```

| 位置 | 含义 |
|------|------|
| 1 | 当前情况/问题核心 |
| 2 | 选项 A 的特点/过程 |
| 3 | 维持现状/不行动 |
| 4 | 选项 B 的特点/过程 |
| 5 | 选择 A 的可能结果 |
| 6 | 不行动的可能结果 |
| 7 | 选择 B 的可能结果 |

### 生命之树 (Tree of Life)

基于卡巴拉生命之树的深度灵性牌阵。

```
          ┌─────┐
          │  1  │ Kether - 神圣意志
          └─────┘
         /       \
    ┌─────┐     ┌─────┐
    │  3  │     │  2  │ Chokmah - 智慧
    └─────┘     └─────┘ Binah - 理解
        \       /
          ┌─────┐
          │  6  │ Tiphareth - 美/和谐
          └─────┘
         /   |   \
    ┌─────┐ ┌─────┐ ┌─────┐
    │  5  │ │  6  │ │  4  │ Chesed - 慈悲
    └─────┘ └─────┘ └─────┘ Geburah - 力量
        \       |       /
          ┌─────┐ ┌─────┐
          │  8  │ │  7  │ Netzach - 胜利
          └─────┘ └─────┘ Hod - 荣耀
              \   /
          ┌─────┐
          │  9  │ Yesod - 基础
          └─────┘
              |
          ┌─────┐
          │ 10  │ Malkuth - 王国/物质
          └─────┘
```

| 位置 | 球体 | 含义 |
|------|------|------|
| 1 | Kether | 最高目标/灵魂使命 |
| 2 | Chokmah | 创造力/男性能量 |
| 3 | Binah | 理解/女性能量 |
| 4 | Chesed | 慈悲/扩展 |
| 5 | Geburah | 力量/限制 |
| 6 | Tiphareth | 核心自我/平衡 |
| 7 | Netzach | 情感/欲望 |
| 8 | Hod | 思想/沟通 |
| 9 | Yesod | 潜意识/基础 |
| 10 | Malkuth | 物质现实 |

## 特殊用途牌阵

### 年度牌阵 (Year Ahead)

用于新年或生日时的年度展望。

```
         ┌─────┐
         │ 13  │ 年度总结
         └─────┘
    
┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐
│ 1 │ │ 2 │ │ 3 │ │ 4 │ │ 5 │ │ 6 │
└───┘ └───┘ └───┘ └───┘ └───┘ └───┘
 1月   2月   3月   4月   5月   6月

┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐
│ 7 │ │ 8 │ │ 9 │ │10 │ │11 │ │12 │
└───┘ └───┘ └───┘ └───┘ └───┘ └───┘
 7月   8月   9月  10月  11月  12月
```

### 马蹄铁牌阵 (Horseshoe)

七张牌的综合分析。

```
┌─────┐                     ┌─────┐
│     │                     │     │
│  1  │                     │  7  │
│     │                     │     │
└─────┘                     └─────┘
  过去                       结果

    ┌─────┐             ┌─────┐
    │     │             │     │
    │  2  │             │  6  │
    │     │             │     │
    └─────┘             └─────┘
    现在                 态度

        ┌─────┐     ┌─────┐
        │     │     │     │
        │  3  │     │  5  │
        │     │     │     │
        └─────┘     └─────┘
        隐藏因素     外部影响
        
              ┌─────┐
              │     │
              │  4  │
              │     │
              └─────┘
              建议
```

| 位置 | 含义 |
|------|------|
| 1 | 过去影响 |
| 2 | 现在情况 |
| 3 | 隐藏因素 |
| 4 | 建议 |
| 5 | 外部影响 |
| 6 | 你的态度 |
| 7 | 可能结果 |

## 牌阵选择指南

### 按问题类型

| 问题类型 | 推荐牌阵 |
|----------|----------|
| 每日指引 | 单牌 |
| 简单问题 | 三牌阵 |
| 事件发展 | 三牌阵（时间线） |
| 关系问题 | 关系牌阵 |
| 重大决定 | 决策牌阵 |
| 全面分析 | 凯尔特十字 |
| 年度规划 | 年度牌阵 |
| 灵性探索 | 生命之树 |

### 按经验水平

| 水平 | 推荐 |
|------|------|
| 初学者 | 单牌、三牌阵 |
| 中级 | 五牌阵、马蹄铁 |
| 高级 | 凯尔特十字、生命之树 |

## 实现配置 (代码)

```typescript
// 牌阵配置类型
interface SpreadConfig {
  id: string
  name: string
  nameEn: string
  cardCount: number
  positions: SpreadPosition[]
  layout: LayoutConfig
  description: string
  suitableFor: string[]
}

interface SpreadPosition {
  index: number
  name: string
  meaning: string
  x: number  // 相对位置
  y: number
}

// 示例：凯尔特十字配置
const celticCross: SpreadConfig = {
  id: 'celtic-cross',
  name: '凯尔特十字',
  nameEn: 'Celtic Cross',
  cardCount: 10,
  positions: [
    { index: 0, name: '现状', meaning: '当前情况的核心', x: 0, y: 0 },
    { index: 1, name: '障碍', meaning: '主要挑战', x: 0, y: 0 }, // 横置
    { index: 2, name: '潜意识', meaning: '潜在影响', x: 0, y: -1 },
    { index: 3, name: '过去', meaning: '近期过去的影响', x: -1, y: 0 },
    { index: 4, name: '意识目标', meaning: '当前的想法和目标', x: 0, y: 1 },
    { index: 5, name: '近未来', meaning: '即将发生的事', x: 1, y: 0 },
    { index: 6, name: '自我态度', meaning: '你对情况的态度', x: 2, y: 2 },
    { index: 7, name: '外部环境', meaning: '他人或环境的影响', x: 2, y: 1 },
    { index: 8, name: '希望与恐惧', meaning: '内心的期望或担忧', x: 2, y: 0 },
    { index: 9, name: '最终结果', meaning: '如果继续当前路径的结果', x: 2, y: -1 }
  ],
  layout: { type: 'celtic', crossCard: 1 },
  description: '最经典的全面分析牌阵',
  suitableFor: ['重大问题', '全面分析', '复杂决策']
}
```
